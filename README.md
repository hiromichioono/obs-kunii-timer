# obs-kunii-timer

OBS Studio 用のストップウォッチスクリプトです。テキストソースに経過時間を `MM:SS` 形式でリアルタイム表示します。

## 特徴

- `os.time()` 差分方式を採用。OBS タイマーの発火遅延に依存しないため、長時間運用でも誤差が蓄積しません。
- キックオフ / 一時停止 / リセットに対応。
- Linux・Mac・Windows のテキストソース両対応（`text_ft2_source` / `text_gdiplus`）。

## 動作環境

- OBS Studio（Lua スクリプト対応バージョン）

## インストール

1. OBS Studio を起動する。
2. メニューから「ツール」→「スクリプト」を開く。
3. `+` ボタンで `timer.lua` を追加する。

## 使い方

1. スクリプトを追加すると設定パネルが表示されます。
2. 「表示するテキストソース」で時間を表示したいテキストソースを選択します。
3. ボタンで操作します。

| ボタン | 動作 |
|--------|------|
| キックオフ / 再開 | タイマーを開始・再開する |
| 一時停止 | タイマーを一時停止する |
| リセット (00:00) | タイマーを停止して `00:00` に戻す |

## 注意

- OBS を再起動すると経過時間はリセットされます。
- タイマーの精度は `os.time()` の 1 秒単位です。
