# obs-kunii-timer

OBS Studio 用のストップウォッチ & YouTube チャットオーバーレイです。

---

## ファイル構成

```
obs-kunii-timer/
├── timer.lua            # OBS Lua スクリプト（タイマー・レガシー）
├── chat_server.py       # YouTube チャット取得・WebSocket サーバー（タイマー制御も兼務）
├── timer_overlay.html   # OBS ブラウザソース用タイマーオーバーレイ
├── timer_control.html   # タイマー操作パネル（ブラウザで開く）
├── chat_overlay.html    # OBS ブラウザソース用チャットオーバーレイ
└── logs/                # チャットログ保存先（.gitignore 対象）
```

---

## チャット & タイマー（chat_server.py + HTML オーバーレイ）

YouTube Live のチャットをリアルタイム表示しながら、ストップウォッチをブラウザ操作できるシステムです。

### 構成

```
YouTube Live
    ↓ pytchat
chat_server.py ──→ WebSocket (ws://localhost:8765) ──→ timer_overlay.html（OBS ブラウザソース）
    ↓                                               ├──→ timer_control.html（手元ブラウザタブ）
logs/...                                            └──→ chat_overlay.html（OBS ブラウザソース）
```

### 動作環境

- Python 3.8 以上
- `pip install pytchat websockets`

### 使い方

**1. サーバーを起動する**

```bash
python chat_server.py "https://www.youtube.com/watch?v=xxxxxx"
```

URL を省略すると入力プロンプトが表示されます。

**2. タイマー操作パネルを開く**

`timer_control.html` をブラウザで開きます（ローカルファイルをダブルクリックするか、ファイル URL で開く）。

| ボタン | 動作 |
|--------|------|
| キックオフ / 再開 | タイマーを開始・再開する |
| 一時停止 | タイマーを一時停止する |
| リセット | タイマーを停止して `00:00` に戻す |
| 時刻を同期 | 分・秒を入力して任意の時刻にセットする |

**3. OBS にブラウザソースを追加する**

| ファイル | 用途 | 推奨サイズ |
|----------|------|------------|
| `timer_overlay.html` | タイマー表示（左下） | 400 × 150 px |
| `chat_overlay.html`  | チャット表示（右下） | キャンバスと同解像度（例: 1920 × 1080） |

OBS でソース追加 → 「ブラウザ」→「ローカルファイル」にチェックを入れてファイルを選択します。

**4. 確認**

- `timer_overlay.html`・`chat_overlay.html` 右下のインジケーターが **LIVE（緑）** になれば接続成功
- タイマーステータス: 動作中は `LIVE`、停止中は `BREAK`（1 秒ごとに点滅）

### 注意

- `chat_server.py` と OBS を同じマシンで起動してください（デフォルト: `ws://localhost:8765`）。
- ログは `logs/` フォルダに自動保存されます（`.gitignore` 対象）。
- `chat_server.py` を再起動するとタイマーはリセットされます。

## タイマー（timer.lua）— レガシー

> **注意:** 現在は上記の WebSocket + HTML 方式を推奨します。`timer.lua` は OBS のテキストソースを直接操作する旧方式です。

OBS のテキストソースに経過時間を `MM:SS` 形式でリアルタイム表示します。

### 特徴

- `os.time()` 差分方式を採用。OBS タイマーの発火遅延に依存しないため、長時間運用でも誤差が蓄積しません。
- キックオフ / 一時停止 / リセット / 時刻同期に対応。
- Linux・Mac のテキストソース対応（`text_ft2_source`）。

### インストール

1. OBS Studio を起動する。
2. メニューから「ツール」→「スクリプト」を開く。
3. `+` ボタンで `timer.lua` を追加する。

### 使い方

1. スクリプトを追加すると設定パネルが表示されます。
2. 「表示するテキストソース」で時間を表示したいテキストソースを選択します。
3. 「ステータスのテキストソース」でステータスを表示したいテキストソースを選択します。
4. ボタンで操作します。

### テキストソースの作成時の注意

- テキストソースのプロパティで **「ファイルから読み込む」のチェックを外す**こと。チェックが入っているとスクリプトからの更新が無効になり、タイマーが表示されません。
- シーンからソースを削除して再追加する場合は「**既存のソースを追加**」を使う。新規作成すると「ファイルから読み込む」がデフォルトで有効になる場合があります。
- スクリプトパネルで **ソース名を選択してから** 操作すること。選択前はタイマー・ステータスともに動作しません。
